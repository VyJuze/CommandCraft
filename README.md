CommandCraft

My project is an interactive web game for learning Minecraft commands. You read the challenge, type the correct command and earn XP as you progress through four categories.

The features are:
Command Reference : study syntax and examples before jumping into challenges
Four categories : Basic Commands, Entities, Movement, and Building
Terminal-style UI : type commands in a console interface with real-time feedback
XP system : earn points for every correct answer
Progress persistence : progress saved per user via Supabase
Auth : email and password authentication

The tech stack that i use was:
React + JavaScript
Tailwind CSS v4
React Router v7
Motion (Framer Motion)
Supabase (Auth + Database)

The requisites to clone the proyect are:
Node.js 22+ and a Supabase project with the progreso table and RLS configured

My proyect (url demo) uses supabase but to clone the repo and run in local i have a file to use localStoradge and no supabase
